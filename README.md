# Rock Paper Scissors

A responsive browser game built with HTML, CSS, and JavaScript. Players can choose moves manually or enable Auto Play for continuous randomized rounds, with scores saved between sessions.

## Features

- Rock, Paper, and Scissors move buttons with image icons
- Random computer move
- Auto Play mode with a start-and-stop toggle
- Win, loss, and tie tracking
- Saved score with `localStorage`
- Reset score button
- Responsive centered layout

- `index.html` 
- `style.css`
